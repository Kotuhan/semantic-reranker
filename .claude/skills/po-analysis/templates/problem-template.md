## Example

**Who is affected?**
Users who need to complete a specific task but face friction in the current flow.

**What problem do they face?**
The current workflow requires too many steps, leading to drop-off before task completion.

**When does this problem occur?**
During the core user journey, especially when users are trying to accomplish their primary goal.

**Why is this problem important NOW?**
User feedback shows 60% of users abandon the flow before completion because the process is too cumbersome.

**What happens if we do nothing?**
Users will continue to churn, and we'll miss our retention targets for Q1.
